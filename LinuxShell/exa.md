# exa
`exa` is a alternative tool for `ls`, which is written in rust

| Command              | Description               |
|----------------------|---------------------------|
| exa --git            | 最终git仓库中新增的文件   |
| exa -T --level=<num> | 按树形结构递归的显示num层 |
| exa -R               | 按非树形结构递归显示目录  |
| exa -1               | 一行显示一项              |
