# ulimit
A tool to get and set user limits

* get limits: `ulimit -<option>`
* set limits: `ulimit -<option> <limit>`

* `-a` list all the limit values
* `-f` file size (blocks)
* `-s` stack size (kbytes)
* `-c` core file size (blocks)
* `-u` processes
* `-n` file descriptors
* `-l` locked-in-memory size (kbytes)
* `-v` address space (kbytes)
